Hubot For Slack With Docker
=======

Simple Hubot Docker build for use in slack. Set the following environment variables and go. 

* `HUBOT_SLACK_TOKEN`

For example:

`> docker run --rm -e HUBOT_SLACK_TOKEN=XXXXXXXXXXXXX shorrockin/hubot-slack-docker`

Other environment variables can be overriden at runtime if desired.

Dockerfile for [shorrockin/hubot-slack-docker](https://registry.hub.docker.com/u/shorrockin/hubot-slack-docker/) image published on Docker Hub.
